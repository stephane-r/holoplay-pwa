# HoloPlay docs

## What is HoloPlay?

HoloPlay is a **Youtube alternative** app using **Invidious API**. You can save music to favoris, create your playlists and import or export your data. This project is fully [open source](https://github.com/stephane-r/holoplay-pwa).
